About this tool

This tool uses dummy data based on the Behavioral Risk Factor Data: Health-Related Quality of Life (HRQOL) dataset from chronicdata.cdc.gov. Visit the dataset page to explore the data in your browser, create custom visualizations, and export the data as CSV, JSON, XML, and/or access it via API.
